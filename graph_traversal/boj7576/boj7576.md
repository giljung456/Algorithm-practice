## BOJ7576 - BFS

- 가변인자 받기

  - \*args

    - 인자를 튜플로 받는다.
    - ```python
      def bfs(*args):
        queue, dist, n, m, store = args;
        //...

      def solution(args):
        n, m, store = args;
        queue = deque();
        bfs(queue, dist, n, m, store);
        // ...
      ```

      단순 나열로 전달한 인자를 하나의 튜플로 받을 수 있다.

    - javascript의 rest문법과 거의 유사하게 사용할 수 있다.

  - \*\*kwargs

    - 인자를 딕셔너리로 받는다.
    - ```python
      def foo(**kwargs):
          print(kwargs);

      foo(color='red', value=1);
      // {'color': 'red', 'value': 1}

      print('foo', end=' ');
      ```

    - 문제풀 때는 굳이 쓸 필요 없을 듯
