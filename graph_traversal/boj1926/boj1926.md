- list comprehension : 간결하게 배열 생성하기
- ```python
  arr = [변수에 대한 표현식 for 변수명 in 이터러블 조건식1 조건식2 ...]
  ```
  필터링을 위한 조건식들은 암묵적으로 and로 연결된다. 명시적으로 and를 써줄 수 없으며 or도 쓸 수 없다.
- ```python

  ```

    board = [list(map(int, sys.stdin.readline().split())) for _ in range(n)];
  return n, m, board; // (1)
